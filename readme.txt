Что может бот:
- Уведомлять о новых постах в добавленных телеграм каналах

Установка бота:
- Установите python на вашу операционную систему
- Создайте бота через телеграм официальный BotFather
- Скопируйте токен вашего бота в телеграм 
- Поставьте свой токен бота в файл token.text
- Запустите бота нажав на start.bat
- Напишите команду /start в своем созданном боте
